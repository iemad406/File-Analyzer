# 🚀 File Path Analyzer Web App

A simple and practical web application that allows users to input a file path and interact with it through a responsive UI and a Python Flask API.

---

## 📌 Overview

This project demonstrates how to build a full-stack web application by connecting a frontend interface with a backend API.

Users can:
- Enter a file path
- Send it to the backend
- Receive and display a structured response

---

## 🛠️ Tech Stack

### Frontend
- HTML5
- CSS3 (Responsive Design)
- JavaScript (Vanilla JS)

### Backend
- Python
- Flask
- Flask-CORS

---

## ⚙️ Features

- ✅ Clean and responsive UI
- ✅ Form validation with error handling
- ✅ API integration using Fetch API
- ✅ JSON request/response handling
- ✅ Scalable project structure
